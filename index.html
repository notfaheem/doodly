About

Doodly is a fun 2d web game where you can play as a character, Doo and interact with various stuff in the game. This game is based on a doodly theme, which is followed mostly throughout the tiny game. And Fun Fact - Most of the pictures u see in this game is hand drawn in MS Paint! Oh.. can we call it hand drawn?

What's in the game

Doodly is a simple game where you can live as a character in the game and do interact with Doo's Home, Market, School, Hospital and more...
