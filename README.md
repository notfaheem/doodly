# Doodly
## Welcome to the Doodly World
Doodly is a fun 2d web game where you can play as a character, Doo and interact with various stuff in the game. 

## What's in the game
Doodly is a simple game where you can live as a character in the game and do interact with :-
- Doo's Home
- Market
- School
- Hospital
- and more

## Why I made Doodly
I am learning coding and doodly is one of my early projects. This is not a big game, it's a simple game, made by me to improve my coding skills.
